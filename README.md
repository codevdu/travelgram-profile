#  Travelgram - Perfil de Viagens

Este é um projeto estático desenvolvido exclusivamente para fins didáticos durante a formação Full-stack da Rocketseat. O objetivo central foi a prática de HTML5 e CSS3, replicando a interface de uma página de perfil para uma rede social fictícia de viagens.

---

## Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias fundamentais da web:

* **HTML5:** Estruturação semântica de dados.
* **CSS3:** Estilização avançada, incluindo:
* **CSS Variables:** Para fácil manutenção de cores e tipografia.
* **Flexbox:** Para alinhamento de componentes e navegação.
* **Importação Modular:** Organização do CSS em múltiplos arquivos para melhor escalabilidade.

* **Google Fonts:** Tipografia personalizada utilizando as fontes *Poppins*, *Alice* e *Open Sans*.
